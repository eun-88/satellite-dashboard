# 🛰️ Global Event-Based Satellite Scheduling Dashboard


## 🛠 Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Map**: Mapbox GL JS
- **Geospatial**: H3 (Hexagonal Hierarchical Geospatial Indexing)
- **Database**: DuckDB, Supabase
- **Data**: GDELT Project (Global Events)

## 🚀 Getting Started
1. `npm install`
2. `.env.local` 파일 생성 후 Mapbox 토큰 입력
3. `npm run dev` 실행
